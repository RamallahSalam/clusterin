Account

/ccn/account[GET]

Ambil data Account berdasarkan user_token :

/ccn/account/search[POST]

Paramater Tipe Data Wajib
user_token string ya

Pendaftaran Account :

/ccn/account/create[POST]

Paramater Tipe Data Wajib
fullname string Ya
address string Ya
whatsapp string Ya
nik string Ya
password string Ya
role enum[RT,Satpam,Warga] Ya
status_account enum[Pending,Active,Inactive] Ya

fullname string Ya
